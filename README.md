# 🍔 Food Delivery App — UI/UX Prototype

A complete, high-fidelity **7-screen mobile app prototype** for a modern food delivery experience, designed as part of my **UI/UX Design Internship at Intern Infobyte**.

The prototype covers the full user journey — from browsing restaurants to tracking a live order — with a consistent design system, reusable components, and an interactive click-through flow.

## 📱 Screens
1. Home — search, categories, offers, popular restaurants
2. Restaurant Menu — category tabs, food cards
3. Food Details — description, ingredients, quantity selector
4. Cart — item management, promo code, price breakdown
5. Checkout — address, payment methods, order summary
6. Order Tracking — live progress timeline, delivery partner info
7. Order Confirmation — success screen with order ID & ETA

## 🎨 Design System
- **Primary color:** Orange `#FF6B00`
- **Neutrals:** White, Light Gray `#F5F5F5`, Dark Gray `#222222`
- **Typography:** Poppins (headings), Inter (body)
- **Style:** 16px rounded corners, soft shadows, consistent 16–24px spacing
